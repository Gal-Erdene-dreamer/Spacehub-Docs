Welcome to Spacehub's documentation!
====================================

**Spacehub** (/spāshəb/) нь харилцагчдад үл хөдлөх хөрөнгөө зайнаас удирдах боломжийг олгодог технологийн компани юм.
Манай компани нь үндсэн мэдээллээ `Spacehub database <https://v2.spacehub.mn/>`_
-ээс татдаг бөгөөд энэ нь *амар хялбар* юм.

:doc:`Менежмент` цэснээс :ref:`Гэрээ` байгуулах, үүсгэх, зэрэг нэмэлт мэдээллийг хараарай.

.. note::

   Энэхүү төсөл нь хөгжүүлэлтийн үе шатандаа явж байна.
   
Spacehub-ийн баримт бичиг нь Read the Docs-ийг ашиглан бүтээгдсэн.
